# 介绍

- 使用原生微信开发者工具进行开发
- 后端使用微信官方提供的云开发
- 教师
  - 可以创建自己的课程并通过系统生成的课程码使得自己的学生加入班级
  - 可以生成一个签到码并进行相关设置进行班级签到
  - 对个人课程进行相关内容设置
  - 查看班级详细信息
- 学生
  - 可以加入班级
  - 进行签到
